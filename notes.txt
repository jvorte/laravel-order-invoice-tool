**Laravel Order & Invoice Automation Tool**

---

## **2. Core Features**

1. **Order Tracking**

   * Παρακολούθηση παραγγελιών (status: pending, paid, shipped, delivered)
   * Store customer info (name, email, phone, address)

2. **Notifications**

   * Email (Laravel Mail / SMTP)
   * WhatsApp (Twilio API ή άλλο provider)
   * SMS (Twilio / Nexmo)
   * Configurable templates για κάθε status

3. **Invoice Generator**

   * Δημιουργία PDF invoices (dompdf)
   * Αποθήκευση σε /storage/invoices
   * Download / Email to customer

4. **Admin Panel**

   * Manage orders, customers, templates
   * View invoices, export CSV/PDF
   * Basic authentication (Laravel Breeze / Jetstream)

5. **Demo User**

   * Username: [demo@example.com](mailto:demo@example.com)

## **3. Tech Stack**

* **Backend:** Laravel 10+
* **Frontend:** Blade + Tailwind CSS (μικρό, καθαρό)
* **Database:** MySQL / SQLite
* **PDF:** dompdf
* **Notifications:** Laravel Mail + Twilio API (SMS / WhatsApp)
* **Authentication:** Laravel Breeze / Jetstream
